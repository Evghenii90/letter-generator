# Letter Generator

Фронтенд-приложение для генерации текста сопроводительного письма и хранения созданных заявок.

## Возможности

- Генерация письма на основе полей формы: `jobTitle`, `company`, `skills`, `details`.
- Предпросмотр результата с индикатором загрузки.
- Сохранение сгенерированных заявок в `localStorage`.
- Отображение списка всех сохраненных заявок на главной странице.
- Копирование текста заявки в буфер обмена.

## Технологии

- React 19
- TypeScript
- Vite
- React Router
- ESLint + Prettier

## Маршруты

- `/` - список заявок (`Applications`)
- `/generations` - форма генерации (`Generators`)

## Локальный запуск

```bash
npm install
npm run dev
```

Обычно dev-сервер Vite запускается на `http://localhost:5173`.

## Доступные скрипты

- `npm run dev` - запуск dev-сервера
- `npm run build` - проверка типов и production-сборка
- `npm run preview` - предпросмотр production-сборки
- `npm run lint` - запуск ESLint
- `npm run lint:fix` - автоисправление lint-ошибок
- `npm run format` - форматирование через Prettier
- `npm run format:check` - проверка форматирования

## Хранение данных

Заявки сохраняются в `localStorage` под ключом:

```txt
messages-generate
```

После перезагрузки страницы данные восстанавливаются автоматически.

## Текущая логика генерации

Сейчас используется локальная шаблонная функция:

`src/shared/lib/utils/generateCoverLetter.ts`

То есть генерация выполняется только на клиенте (без внешнего AI API).

## Примечания

- Поле `details` поддерживает до `1200` символов.
- В интерфейсе показывается прогресс по первым 5 заявкам; при этом создавать больше 5 заявок также можно.
